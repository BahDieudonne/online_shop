# CHANCELOR STORE 🛍🛒🇨🇲

> Enterprise-grade e-commerce platform for Cameroon's leading online store.

## Stack
- **Frontend**: React 18 + Vite + Redux Toolkit + Tailwind CSS (PWA)
- **Backend**: Node.js + Express.js
- **Database**: MongoDB + Mongoose
- **Auth**: JWT + Refresh Tokens + RBAC
- **Payments**: Stripe, PayPal, MTN Mobile Money, Orange Money, Bank Transfer, COD
- **Storage**: Cloudinary / AWS S3 / Google Cloud (env-switchable)
- **Infra**: Docker + Nginx + PM2

## Quick Start
```bash
# Backend
cd backend && npm install && cp .env.example .env
npm run dev

# Frontend (new terminal)
cd frontend && npm install && cp .env.example .env
npm run dev
```

## Admin Panel
URL: `/admin`
Default: `admin@chancelorstore.cm` / `Admin@123!`

## Features
- Amazon-grade search with autocomplete
- Product variants (color, size, storage, etc.)
- Flash sales with countdown timers
- MTN & Orange Money payments (Cameroon)
- Real-time order tracking
- Loyalty points & referral program
- Full admin dashboard with analytics
- PWA with push notifications
- Bilingual (French/English) Cameroon market
